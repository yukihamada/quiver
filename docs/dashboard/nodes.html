新しいノードを追加

  1. QUIVer Nodeをインストール

    お使いのOSに応じてインストールしてください

    Linux / macOS:
    curl -fsSL https://quiver.network/install.sh | sh
    Windows:
    irm https://quiver.network/install.ps1 | iex

  2. プロバイダトークンを設定

    以下のトークンをコピーして設定してください

    provider_pk_1234567890abcdef

  3. ノードを起動

    quiver-node start --name "my-gpu-node"

  4. 動作確認

    ノードが正常に起動すると、このダッシュボードに自動的に表示されます。
