技術統合の仕組み

QuicPairからQUIVerへの切り替えは、APIレベルで透過的に実装されています。

// QuicPair内での処理スコープ判定
const result = await quicpair.process({
  prompt: "大規模な文書を要約して",
  scope: "auto" // 自動判定
});

// 内部で自動的に：
// 1. ローカルリソースをチェック

// 3. 必要に応じてQUIVerを提案
// 4. ユーザー承認後、透過的に実行
