QUIVerプラットフォームの構成

🌐 QUIVer = プラットフォーム全体

QUIVerは「完全プライベートAIネットワーク」を実現するための基盤プロジェクトです。 P2P通信、暗号化、計算レシート、決済などの仕様を定義し、 その上で様々なアプリケーションが動作します。

📱 QuicPair = 最初のアプリ

QuicPairはQUIVerプラットフォーム上で動作する最初のアプリケーションです。 Mac ↔ iPhoneの超高速ペアリングと完全ローカルAIチャットを提供します。 WebRTC/QUIC/NoiseIKによるE2E暗号化通信を実装しています。
